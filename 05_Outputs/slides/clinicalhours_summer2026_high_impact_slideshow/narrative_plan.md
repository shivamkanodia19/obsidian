# ClinicalHours Summer 2026 High-Impact Slideshow Narrative Plan

## Audience

Students trying to line up clinical hours for Summer 2026 without bouncing between school docs, screenshots, extra tabs, and scattered clinic pages.

## Objective

Create a shorter, more visual, higher-impact social slideshow that still stays inside truthful ClinicalHours student proof.

## Narrative Arc

1. Summer hook and scattered search problem
2. One-map structural payoff
3. One-real-result proof
4. Same-result save-and-follow-through proof
5. Low-friction CTA

## What changed from the earlier faceless deck

- Fewer decorative chips and fewer repeated bottom labels
- More screen space given to hero proof assets
- Tighter middle-story continuity around one real Houston Methodist example
- Shorter copy built for sound-off social viewing
- Stronger layout variation so each slide feels like a distinct step, not one template reused five times

## Platform intent

- Base format: `1080 x 1920`
- Primary use: TikTok and Instagram Reel-style slideshow posting
- Secondary use: individual slide export for static testing
- Editing intent after export: quick timing, subtle zooms, sound-off legibility, and light music or voiceover layered in the posting tool

## Testing note

TikTok's official carousel ad playbook says `3` or `7-9` images often perform better than other counts, but this build stays at `5` slides because the ClinicalHours proof chain needs a clean `problem -> map -> result -> save -> CTA` sequence.

Recommended follow-up tests:

- `3`-slide cutdown for colder paid traffic
- `7`-slide expansion if we later add verified student testimonials or additional public proof
