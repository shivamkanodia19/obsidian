# ClinicalHours Summer 2026 Premium Problem-First Slideshow Narrative Plan

## Audience

Pre-health students who know they need clinical hours but feel stuck between scattered searches, unclear paths, and limited time.

## Objective

Create a more aesthetic, more emotional, less informational ClinicalHours slideshow for TikTok and Instagram that behaves like a premium short-form campaign instead of a student explainer.

## Narrative arc

1. `SEARCH` - everyone says get clinical hours
2. `ACCESS` - the path in still feels unclear
3. `TIME` - the good openings move before the student does
4. `START` - ClinicalHours becomes the first calm, credible starting point
5. `SEE` - one real opportunity proves usefulness
6. `SAVE` - low-pressure browse/save CTA

## Messaging rules

- fewer words
- one promise per slide
- no literal product screenshots in slides `1-3`
- screenshots only after the problem and stakes are established
- CTA stays soft and reversible

## Publishing intent

- Base format: `1080 x 1920`
- Primary use: TikTok and Instagram portrait slideshow / Reel assembly
- Intended pacing: `10-15` seconds total if animated in posting software
- Motion recommendation: subtle zooms only
- Audio assumption: sound-off first

## Follow-up tests

- `5`-slide cut for paid awareness
- `7`-slide cut if later adding student proof or testimonial layers
- second aesthetic branch using the same proof but a more minimalist still-life opening
