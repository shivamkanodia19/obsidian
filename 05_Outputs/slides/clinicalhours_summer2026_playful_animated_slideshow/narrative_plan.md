# ClinicalHours Summer 2026 Playful Animated Slideshow Narrative Plan

## Audience

Pre-health students who need clinical hours but feel like the search is scattered, slow, and hard to trust.

## Objective

Keep the stronger problem-first structure, but make the deck more playful, more projected, and less severe by using a recurring animated student character in the first half.

## Narrative arc

1. `SEARCH` - need hours, not a scavenger hunt
2. `MAYBE` - every site says maybe
3. `TIME` - the good ones move fast
4. `START` - start with what's real
5. `SEE` - see what counts
6. `SAVE` - browse first, save the good ones

## Character rules

- one recurring animated pre-health student
- simplified face
- broad audience fit
- playful but not childish
- no literal product screenshots until slide `4`

## Messaging rules

- one main line per slide
- no informational paragraph copy
- proof stays concentrated in the back half
- CTA stays low-pressure

## Publishing intent

- Base format: `1080 x 1920`
- Primary use: TikTok and Instagram slideshow / Reel assembly
- Audio assumption: sound-off first
- Motion recommendation: subtle zoom or parallax only

## Follow-up tests

- export a `5`-slide paid-social cut
- test alternate character variants across campaigns
- build a Reel version with light motion and timing
