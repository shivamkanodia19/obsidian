# ClinicalHours TikTok Review Bundles

This folder stores JSON review packets emitted by the render script.

Each bundle should describe:

- the rendered suffix
- source HTML file
- export paths
- QA screenshot paths
- slide-level QA text
- dimensions

Use these files as the default cold-review handoff packet for fresh-context judges.
