# ClinicalHours TikTok Captures

Real product captures and supporting notes used as source material for the slideshow.

## How To Use This Folder

- Read the replication contract first: [[05_Outputs/images/clinicalhours_tiktok_2026-05-18/docs/playbooks/clinicalhours_tiktok_replication_playbook_2026-05-19]]
- Treat enterprise element-level recaptures as the strongest current `clinic_ops_primary` proof.
- Treat the student homepage lower stat cards as invalid proof because the public capture shows placeholder `0+` metrics.
- Treat `official/` as the strongest current student search-result and next-step proof lane when a branch needs one coherent public example instead of generic dashboard support.

## Sections

- `auth/` - authenticated dashboard and sign-in-state screenshots
- `derived/` - canonical crops used by the bright branch and other layout-specific proof treatments
- `enterprise/` - enterprise page screenshots and cropped proof sections
- `home/` - homepage screenshots across desktop and mobile
- `map/` - map experience screenshots and panel states
- `official/` - promoted live captures used for the semester-start proof chain and other direct student-proof repairs
