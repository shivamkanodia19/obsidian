# Local SEO

Optimization for local search visibility.

## Google Business Profile

### Setup Essentials
- Verify ownership
- Complete all fields
- Choose accurate categories
- Add photos regularly
- Post updates weekly

### Optimization
- Business description with keywords
- Products/services listed
- Hours updated
- Attributes selected
- Q&A populated

## Local Citations

### NAP Consistency
- Name exactly as registered
- Address with suite/unit
- Phone with area code
- Consistent across all listings

### Priority Directories
- Google Business Profile
- Bing Places
- Apple Maps
- Yelp
- Industry-specific directories

## Local Keywords

### Keyword Patterns
- [service] + [city]
- [service] + near me
- best [service] + [city]
- [service] + [neighborhood]

### On-Page Local Signals
- City/region in title tags
- Local schema markup
- Location pages for multi-location
- Embedded Google Map

## Review Management

### Getting Reviews
- Ask satisfied customers
- Make it easy (direct link)
- Timing: after positive experience
- Follow up via email

### Responding to Reviews
- Respond to all reviews
- Thank positive reviewers
- Address negative professionally
- Take issues offline when needed

## Local Schema Markup

```json
{
  "@type": "LocalBusiness",
  "name": "Business Name",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "123 Main St",
    "addressLocality": "City",
    "addressRegion": "State",
    "postalCode": "12345"
  },
  "telephone": "+1-555-555-5555",
  "openingHours": "Mo-Fr 09:00-17:00"
}
```

## Local Ranking Factors

1. Google Business Profile signals
2. Link signals (local relevance)
3. Review signals (quantity, quality)
4. On-page signals (NAP, keywords)
5. Citation signals (consistency)
6. Behavioral signals (clicks, check-ins)
