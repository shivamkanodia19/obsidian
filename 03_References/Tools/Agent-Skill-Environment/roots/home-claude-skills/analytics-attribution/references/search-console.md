# Google Search Console

SEO performance monitoring and optimization.

## Key Reports

### Performance Report
- Total clicks
- Total impressions
- Average CTR
- Average position

### Filters
- Query (keywords)
- Page (URLs)
- Country
- Device
- Search type
- Date range

## Analysis Framework

### Query Analysis
- Which keywords drive traffic?
- CTR by position
- Impressions without clicks
- New vs existing queries

### Page Analysis
- Top performing pages
- Underperforming pages
- Cannibalization issues
- Content gaps

## Index Coverage

### Status Types
| Status | Meaning |
|--------|---------|
| Valid | Indexed and shown |
| Warning | Indexed with issues |
| Excluded | Not indexed |
| Error | Can't be indexed |

### Common Issues
- Crawl anomalies
- Redirect errors
- 404 pages
- Noindex tags
- Duplicate content

## Core Web Vitals

### Metrics
- LCP: Loading performance
- FID: Interactivity
- CLS: Visual stability

### Assessment
- Good (green)
- Needs Improvement (yellow)
- Poor (red)

## Links Report

### External Links
- Top linked pages
- Top linking sites
- Top linking text

### Internal Links
- Top linked pages
- Link distribution
