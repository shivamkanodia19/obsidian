# Engagement Tactics

Strategies for building social media engagement.

## Hook Formulas

### Curiosity Hooks
- "Most people don't know this..."
- "Here's what no one tells you about..."
- "I tested X for 30 days. Results:"

### Controversial Hooks
- "Unpopular opinion:"
- "Hot take:"
- "I disagree with the common advice to..."

### Story Hooks
- "Last year I was [problem]. Here's what changed:"
- "I made a $X mistake so you don't have to"
- "The moment I realized..."

### Value Hooks
- "X tips to [achieve result]"
- "How I [achieved result] in [timeframe]"
- "The framework I use to..."

## Comment Engagement

### Responding Strategy
- Respond within first hour
- Ask follow-up questions
- Add value in responses
- Pin best comments

### Encouraging Comments
- End posts with questions
- Ask for opinions
- Request "drop a [emoji] if..."
- Create debate opportunities

## Community Building

### Tactics
- Create consistent series
- Build rituals (weekly posts)
- Spotlight community members
- Create inside jokes/language
- Host AMAs and Q&As

### User-Generated Content
- Create shareable templates
- Run challenges
- Feature customer content
- Create brand hashtags

## Influencer Collaboration

### Finding Partners
- Micro-influencers (10K-100K)
- Aligned audience
- Authentic engagement
- Content quality match

### Collaboration Types
- Content swaps
- Takeovers
- Co-created content
- Product features
- Affiliate partnerships

## Engagement Metrics

| Metric | Good | Great |
|--------|------|-------|
| Engagement Rate | 1-3% | 3-6%+ |
| Comment Rate | 0.1% | 0.5%+ |
| Share Rate | 0.5% | 1%+ |
| Save Rate | 1% | 3%+ |
