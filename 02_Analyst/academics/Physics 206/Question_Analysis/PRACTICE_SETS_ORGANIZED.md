# Practice Sets: Organized by Topic & Difficulty

Complete reference guide to find and practice specific question types across all 15 exams.

## HOW TO USE THIS GUIDE

**Pick a practice strategy:**

1. **By Difficulty** (Recommended for first 2 weeks)
   - Start with "All Easy" problems
   - Progress to "All Easy-Medium"
   - Then "All Medium"
   - Use YEAR_BY_YEAR_BREAKDOWN.md to find specific problems

2. **By Topic** (Recommended for targeted learning)
   - Master one topic at a time
   - Do all problems on that topic across different exams
   - Compare how topics vary by year

3. **By Year** (Recommended for final exam prep)
   - Simulate full 50-minute exams
   - Use actual exam PDFs from Exam3/ folder
   - Time yourself on Problems 1-4

4. **By Difficulty + Topic** (Most targeted)
   - "All Easy Collision Problems"
   - "All Medium-Hard Angular Momentum Problems"
   - Maximum targeted learning

## GUARANTEED PROBLEMS BY TOPIC

### TOPIC 1: POLAR KINEMATICS (100% Frequency)

**Problem 1 from EVERY exam (2005-2019)**
- 15 problems total
- All 25 pts each
- All same format: Derive $v_r$, $v_\theta$, $a_r$, $a_\theta$

**Practice Levels:**

| Difficulty | Exams to Practice | Time | Strategy |
|------------|------------------|------|----------|
| Easy-Medium | 2007, 2008, 2009, 2015, 2018, 2019 | 8-10 min each | Learn standard format |
| Easy-Medium | All 15 exams | 8-10 min each | Master derivation |

**Key Formulas to Know:**
- $v_r = \dot{r}$
- $v_\theta = r\dot{\theta}$
- $a_r = \ddot{r} - r\dot{\theta}^2$
- $a_\theta = r\ddot{\theta} + 2\dot{r}\dot{\theta}$

**Time Allocation:** 5 minutes on exam (get it right = 25 pts guaranteed)

---

### TOPIC 2: COLLISIONS & MOMENTUM (100% Frequency)

**Appears in:** Problem 2 (all exams except some shifts)
- ~15 problems across all years
- Points: 12-25 per exam

**Collision Types:**

#### Inelastic (Objects Stick)
- Momentum conserved
- Energy lost to deformation
- Formula: $m_1 v_1 + m_2 v_2 = (m_1 + m_2)v_f$

**Exams with inelastic collisions:**
- 2007 Problem 2a (12 pts, medium) - ball swings + collision
- 2009 Problem 2b (13 pts, medium) - object sticks to block
- Others in 2005-2006, 2011-2019

#### Elastic (Objects Bounce)
- Momentum AND energy conserved
- Two equations for two unknowns

**Exams with elastic collisions:**
- 2007 Problem 2b (13 pts, easy-medium) - set up equations
- 2009 Problem 2a (12 pts, medium) - with springs
- Others throughout all years

#### Multi-Dimensional Collisions
- Conserve momentum in each direction separately
- Track velocity vectors

**Exams with 2D collisions:**
- 2008 Problem 2b (13 pts, medium-hard)

**Practice Path:**

| Week | Focus | Exams | Notes |
|------|-------|-------|-------|
| 1 | Inelastic basics | 2007/2a, 2009/2b | Understand momentum + energy loss |
| 1-2 | Elastic setup | 2007/2b, 2009/2a | Learn two-equation system |
| 2 | Multi-dimensional | 2008/2b | Add vector complexity |
| 3+ | Full mix | All collision problems | Timed practice |

### TOPIC 3: ANGULAR MOMENTUM & CONSERVATION (100% Frequency)

**Appears in:** Problem 3 (all exams)
- ~15 problems total
- Points: 8-25 per exam

**Core Concept:** $L = I\omega$ = constant (when no external torques)

**Common Scenarios:**

#### Moment of Inertia Changes (Very Common)
- Person pulls arms in
- Mass moves toward/away from axis
- As I decreases, omega increases

**Exams:**
- 2007 Problem 3b-c (16 pts total, easy-medium + medium)
- 2008 Problem 3a-c (24 pts total, medium-hard)
- All 2005-2019 exams likely include

**Key Insight:** $I_i \omega_i = I_f \omega_f$ (solve for unknowns)

#### Angular Momentum About a Point
- $L = \vec{r} \times m\vec{v}$
- Plane or object moving in space

**Exams:**
- 2008 Problem 4a (6 pts, easy-medium) - electron orbit
- 2009 Problem 4a (5 pts, easy) - plane's angular momentum

#### Composite Systems
- Rod + masses moving together
- Calculate total moment of inertia

**Exams:**
- 2007 Problem 3c (8 pts, medium)
- 2008 Problem 3c (8 pts, medium-hard)
- 2009 Problem 4b (7 pts, medium)

**Moment of Inertia Reference:**

| Shape | Formula | Context |
|-------|---------|---------|
| Point mass at distance r | $I = mr^2$ | Single mass on rod |
| Solid cylinder | $I = \frac{1}{2}MR^2$ | Wheel, disk |
| Thin hoop | $I = MR^2$ | Ring, all mass at edge |
| Thin rod (about center) | $I = \frac{1}{12}ML^2$ | Rod spinning at midpoint |
| Thin rod (about end) | $I = \frac{1}{3}ML^2$ | Rod rotating at one end |

**Practice Path:**

| Week | Focus | Exams | Notes |
|------|-------|-------|-------|
| 1-2 | I changes | 2007/3b-c | Master figure-skater effect |
| 2 | About a point | 2008/4a, 2009/4a | Learn cross product method |
| 2-3 | Composite I | 2008/3c, 2009/4b | Add multiple parts |
| 3+ | Full mix | All angular momentum problems | Timed practice |

### TOPIC 4: TORQUE & ROTATIONAL DYNAMICS (100% Frequency)

**Appears in:** Problem 4 (all exams)
- ~15 problems total
- Points: 6-25 per exam

**Core Equations:**
- Torque: $\tau = rF\sin\theta = r_\perp F$
- Newton's 2nd (rotation): $\tau = I\alpha$

**Common Scenarios:**

#### Rod Rotating About Hinge
- Gravity acts at center (L/2)
- Different forces at different angles

**Exams:**
- 2007 Problem 4 (25 pts total, 4 sub-parts)
  - 4a: Horizontal position (6 pts, easy)
  - 4b: Horizontal acceleration (6 pts, easy-medium)
  - 4c: Vertical position (6 pts, easy)
  - 4d: Vertical acceleration (7 pts, hard)

**Key Insights:**
- Torque = 0 when force passes through axis
- Torque = 0 when force parallel to radius
- Tangential acceleration: $a = r\alpha$
- Centripetal acceleration: $a_c = \omega^2 r$

#### Conveyor Belts & Circular Paths
- Both tangential and centripetal forces
- Friction constraints

**Exams:**
- 2009 Problem 3 (25 pts, easy-medium + medium-hard)
  - 3a: Velocity at point (8 pts)
  - 3b: Force vs. time (8 pts)
  - 3c: Max acceleration (9 pts)

**Practice Path:**

| Week | Focus | Exams | Notes |
|------|-------|-------|-------|
| 1-2 | Rod problems | 2007/4a-d | Learn torque basics |
| 2 | Conveyor belt | 2009/3a-c | Add circular motion complexity |
| 2-3 | Mixed torque | Exams with pulley systems | Compare scenarios |
| 3+ | Full mix | All rotation problems | Timed practice |

## EXAM DAY STRATEGY

### Time Allocation (50 minutes total)

**Problem 1: Polar Kinematics (10 minutes)**
- Guaranteed 25 points
- Don't overthink
- Standard format every time
- Time: 5-8 minutes (leave 2-3 min buffer)

**Problems 2-4: Variable (40 minutes total)**
- Aim for 60+ pts to reach 85 total
- Problems 2 & 3: ~15 mins each (collisions + angular momentum)
- Problem 4: ~10 mins (rotational dynamics)

### Strategy

1. **Start with Problem 1** (5 mins) - Get those 25 pts
2. **Read Problems 2-4** (2 mins) - Pick your best topics
3. **Solve in order** (40 mins) - Don't skip ahead
4. **Leave 3 mins** - For review/corrections

### If Short on Time

- Problem 1: Always complete (25 pts = half the exam)
- Problem 2: Next priority (12-25 pts) - Usually collision/momentum
- Problems 3-4: Do as much as possible
