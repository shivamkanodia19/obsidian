# CSCE 120 — Computer Programming (C++)

## Course Overview

TAMU honors-track C++ introductory programming course. Focus on foundational memory management, OOP, and system-level programming concepts.

## Topics Covered

- **Dynamic Memory Management** — new/delete, memory allocation strategies
- **Rule of Three** — constructor, copy constructor, destructors (now Rule of Five in modern C++)
- **Operator Overloading** — custom operators for user-defined types
- **Pointer Mechanics** — address-of (&), dereference (*), pointer arithmetic
- **Dynamic Arrays** — manual array management, resizing strategies
- **Input/Output Streams** — cin, cout, file I/O, stream manipulation
- **Exception Handling** — try/catch/throw, custom exceptions

## Learning Outcomes

- Understand memory model (stack vs. heap)
- Implement classes with proper resource management
- Debug pointer-related errors
- Work with dynamic data structures (arrays, lists)
- Apply OOP principles in C++
