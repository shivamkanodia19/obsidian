## Troubleshooting

### What if emails bounce?
→ See: `followup_sending_schedule_2026-04-24.md` > Bounce Handling Procedure
→ Do: Find alternative contact, retry within 3 days

### What if response rate is <15%?
→ See: `followup_angles_abtest_framework.md` > Response Quality Rubric
→ Analyze: Which angles underperformed? Try different subject lines?

### What if we get 0 positive responses?
→ Unlikely, but if it happens: Check email deliverability, verify content resonates, consider phone outreach instead

### What if a clinic asks for pricing immediately?
→ See: `FOLLOWUP_BATCH_README.md` > Common Questions
→ Do: Send pricing based on volunteer count estimate, emphasize 30-day trial option

### What if I miss the April 24 send date?
→ That's OK. Extend timeline: Send April 25-26 instead
→ Adjust all downstream dates (response window, analysis, etc.) by same offset
