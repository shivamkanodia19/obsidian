# Speculative High-Upside Screen - 2026-05-12

Scope: backfilled from unsaved Codex stock sessions from `2026-05-11` through `2026-05-12`.

This note preserves a dated speculative screen and keeps the scopes separate instead of flattening them into one noisy watchlist.

## Scope Split

- `Explosive upside screen`: names that could plausibly `3x` from a strong catalyst path.
- `Risk-sanity screen`: names that still pass a stricter position-sizing and durability filter.

A name can pass the first screen and fail the second. That is a scope difference, not automatically a contradiction.

- `AEVA`

## Durable Takeaway

- If the goal is actual sized speculative exposure, start from the `risk-sanity` list first: `CLSK`, `CRDO`, `COIN`, `IREN`.
- If the goal is pure option-like upside, the `3x` screens were much looser and included names the risk pass would not size normally.
- This note is a dated snapshot, not a current recommendation. Any reuse should start with a fresh live-data pass.
