# Wave 4 Tier 2 — Validation Checklist (LinkedIn DM Before Sending)

**Status:** ⏳ 5 companies require LinkedIn DM validation before sending cold emails

**Instructions:**
1. Send LinkedIn DM to decision-maker with validation question
2. Wait 24-48 hours for response
3. If confirms problem: Send cold email (copy-paste from READY-TO-SEND-TIER-2.md)
4. If denies or no response: Skip or defer

---

## VALIDATION FLOW

```
Send DM → Wait 24-48 hours → Response?
  ├─ YES (confirms) → Send email
  ├─ NO (denies) → Skip/defer
  └─ No response → Skip/defer (low priority)
```
