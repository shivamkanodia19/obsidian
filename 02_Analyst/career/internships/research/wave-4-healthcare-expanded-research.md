## Research Execution Workflow

For each facility (1-5):

1. **CMS Data Pull (5 min per facility):**
   - Visit https://www.medicare.gov/care-compare/
   - Search facility name + "DFW" or zip code
   - Record: Deficiency count, fines, staffing turnover rates
   - Copy facility address + phone

2. **ProPublica Cross-Check (3 min per facility):**
   - Visit https://projects.propublica.org/nursing-homes/
   - Search facility name
   - Verify turnover data + deficiency patterns
   - Document turnover % for RNs vs. nursing staff

3. **Glassdoor Employee Review Search (5 min per facility):**
   - Glassdoor search: "[Facility Name] [City]"
   - Look for: "compliance", "short-staffed", "turnover", "chaos", "burnout"
   - If 3+ reviews mention staffing/compliance issues = strong confidence
   - Document 2-3 quotes

4. **LinkedIn Decision-Maker Search (5 min per facility):**
   - LinkedIn: Search "[Facility Name]" + "Administrator" or "Executive Director"
   - Record: Name, title, LinkedIn URL
   - If not found on facility page, search company + title

5. **Website Contact Info (2 min per facility):**
   - Visit facility website
   - Find: Administrator name, phone, email (if available)
   - Look for "Contact" or "About" section

**Total per facility:** 20 minutes  
**Total for 5 facilities:** ~1.5-2 hours
