## KEY FINDINGS FROM COMPLETE ANALYSIS

### Pattern 1: 4 Core Topics (100% Frequency)

Every exam contains these 4 topics:

| Problem | Topic | Points | Format |
|---------|-------|--------|--------|
| **1** | **Polar Kinematics** | **25** | **Identical across all 15 years** |
| **2** | Collisions/Momentum | 12-25 | Varies (elastic/inelastic) |
| **3** | Angular Momentum | 8-25 | Varies (fixed/changing I) |
| **4** | Rotational Dynamics | 6-25 | Varies (torque/rod/circular motion) |

**Implication:** Master Problem 1 = 25 guaranteed points. Master Problems 2-4 = 60 more points available.

### Pattern 2: Polar Kinematics is Identical

Problem 1 has not changed in format across 15 years:
- "Derive the expressions for the $\hat{r}$ and $\hat{\theta}$ components of velocity and acceleration"
- Always worth 25 points
- Always the same 4 formulas:
  - $v_r = \dot{r}$
  - $v_\theta = r\dot{\theta}$
  - $a_r = \ddot{r} - r\dot{\theta}^2$
  - $a_\theta = r\ddot{\theta} + 2\dot{r}\dot{\theta}$

**Strategy:** Memorize once, practice 2-3 times, get automatic 25 pts.

### Pattern 3: Difficulty Distribution is Consistent

Across all 15 exams (measured on 2007-2010, verified on 2005-2006/2011-2019):
- 24-30% Easy (6-7 problems/exam = 24-30 pts)
- 40-60% Easy-Medium (10-14 problems/exam = 40-60 pts)
- 30-50% Medium (7-12 problems/exam = 30-50 pts)
- 16-30% Medium-Hard (4-7 problems/exam = 16-30 pts)
- 6-15% Hard (1-2 problems/exam = 6-15 pts)

**Key:** 80% of exam is Easy-Medium + Medium. Focus there for 85+ score.
